# 🧠 Neuroscience: Advanced Brain Data Analysis
**By Gladys Choque Ulloa** *Ph.D. Student in Computer Science (USP) | Statistics & Data Science Specialist | Machine Learning | Deep Learning *

---

## 📌 Overview
Bienvenido a mi repositorio central de **Neurociencia Computacional**. Aquí documento investigaciones, modelos de aprendizaje profundo y análisis estadísticos aplicados al estudio del cerebro. Mi enfoque combina el rigor de la estadística académica con la potencia del **Deep Learning** para resolver problemas complejos en neuroimagen.

## 🔬 Core Projects
En este repositorio encontrarás implementaciones enfocadas en:

* **3D Brain Segmentation:** Segmentación automatizada de estructuras cerebrales mediante arquitecturas U-Net y modelos Encoder-Decoder.
* **Medical Image Processing:** Pipelines para el manejo de archivos NIfTI (.nii), normalización de intensidades y transformaciones espaciales 3D.
* **Statistical Neuroimaging:** Análisis de distribución de intensidades y mapeo estadístico cerebral.

## 🛠️ Tech Stack
* **Deep Learning:** TensorFlow / Keras
* **Neuroimaging:** Nibabel, Nilearn
* **Data Analysis:** NumPy, SciPy, Pandas, Matplotlib
* **Environment:** Google Colab / Jupyter Notebooks

## 📊 Methodology Focus
Cada análisis sigue un flujo de trabajo científico:
1.  **Ingesta de Datos:** Manejo de volúmenes médicos multidimensionales.
2.  **Visualización Multiaxial:** Inspección en planos Axial, Sagital y Coronal.
3.  **Preprocesamiento:** Normalización y optimización de datos para modelos de IA.
4.  **Validación:** Evaluación de precisión mediante mapas de error y métricas de segmentación.
